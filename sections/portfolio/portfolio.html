K! Learn

Kazza Learn is a quiz application completly open source! It uses react on the hood and has various learning modes to choose. This app supports offline use, custom questions-awnsers, endpoints for creating custom modes and local progress save using localstorage.

Imprint

Imprint is a chinese-inspired comment board for visitors to leave real-time notes. Fun and interactive way of having expressing the freedom of speech. Open-source project done with React, Firebase, Tailwind CSS.
